# kadode_fine_tuning

かどで日記に最適化されたモデルを作るための検証リポジトリ

# 環境構築

poetry を使用

新規で作る

```
poetry new 名前
```

セットアップ

```
poetry install
```

パッケージ追加

```
poetry add パッケージ名
```

パッケージ更新

```
poetry update
```

実行

```
poetry run python ファイル名
```

or

```
poetry shell
python ファイル名
```
