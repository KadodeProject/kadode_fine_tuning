# kadode_fine_tuning

かどで日記に最適化されたモデルを作るための検証リポジトリ.

GPT2によりpre trainingにrinna/japanese-gpt2-medium, fine tuningにusuyukiの日記を用いて独自モデルを作成して検証した結果, usuyuki個人の日記のみでは**独自モデルによるかどで日記の機能価値向上は難しい**という結論に至った。
